## 설명
이 디렉터리는 Linkly Calendar API 서버(NestJS) 프로젝트입니다.
기존 NestJS 기본 템플릿 안내를 프로젝트 환경에 맞게 한국어로 정리했습니다.

## 설치
```bash
pnpm install
```

## 실행
```bash
# 개발
pnpm run start

# 감시 모드
pnpm run start:dev

# 운영 모드
pnpm run start:prod
```

## 테스트
```bash
# 단위 테스트
pnpm run test

# e2e 테스트
pnpm run test:e2e

# 커버리지
pnpm run test:cov
```
